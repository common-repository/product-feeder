== Description ==

Connect to the marketplace(s) of your choice to increase sales, synchronize orders & returns, rule-based product selection, and much more! Try it now at https://product-feeder.com

== Installation ==

1. Install and activate our plugin.
2. Create a account at https://product-feeder.com
3. Create a Wordpress WooCommerce source by using the information shown by the plugin
4. Start connecting the marketplace(s) of your choice!
